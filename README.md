# Pritelesh Bhowmik Nel - Creative Portfolio

A modern, high-end creative portfolio featuring neumorphic design, cinematic carousels, and smooth animations. Built for performance and visual impact.

## 🚀 Live Demo
Deploy this project on [Vercel](https://vercel.com) for the best experience.

## ✨ Features
- **Cinematic Showcase**: High-end project gallery with 3D depth and focus effects.
- **Neumorphic UI**: Tactile, soft UI design with realistic shadows and micro-interactions.
- **Dynamic Skills & Experience**: Interactive sections highlighting technical expertise and professional history.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern Animations**: Powered by Framer Motion for a fluid, premium feel.

## 🛠️ Tech Stack
- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/pritels-creative-hub.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Deployment
This project is optimized for **Vercel**. Simply push your code to GitHub and connect the repository to Vercel. It will automatically detect Vite and deploy your site.
